    Aprendizaje de las Etiquetas Semánticas, Listas y Enlaces

    El objetivo principal del uso de las estiquetas, listas y enlaces es crear un sitio web con un contenido más util, mejor formado y que sea facil de procesar por cualquier sistema informático automatizado. Su uso es bastante simple e intuitivo y no existe una norma fija, si no que cada diseñador puede darle el uso que considere más apropiado.

    El estandar HTML5 permite especificar que marcas deben utilizarsen para escribir un documento y que significada cada una de ellas, permitiendo que cualquier navegador pueda interpretarlas. Un ejemplo claro es que todo docuemento HTML debe ir delimitado por la etiqueta < html > de inicio y de cierre , dentro de estas etiquetas esta la cabecera < head > y el cuerpo < body >
    Estas etiquetas son las vamos a encontrar en cualquier pagina web y que permite definir su estructura.


    Imagen extraida de la pagina https://www.aulaclic.es/articulos/html5-semantica_1.html "posibles formas de ordenar un documento con las etiquetas"

    Las etiquetas semánticas son aquellas que dan un significado a las partes del documento

    La estructura de un documento HTML es un lenguaje de marcas, que consta de texto y define el contenido real de una página web, el cual debe contener marcas especiales conocidas como las etiquetas que permiten dar un significado al texto o contenido de la misma.
    Una etiqueta se clasifica como semántica, sí sirve para indicar que son y cual es el significado de los elementos que contienen adentro. Estas han ganado mucha importancia en el marco del HTML y de la composición de un documento web, ayudando a los motores de busqueda como Google a indexar de forma correcta los contenidos de un sitio.
